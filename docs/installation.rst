============
Installation
============

Install ``python_secrets`` using the ``pip`` module of Python 3.6 (or greater):

.. code-block:: console

   $ python3 -V
   $ Python 3.6.6
   $ python3 -m pip install python_secrets

..

..

For best results, use a Python ``virtualenv`` to avoid problems due to
the system Python not conforming to the version dependency.
